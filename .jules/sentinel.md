## 2024-04-14 - Fix weak random generation in invite codes\n**Vulnerability:** Weak random number generation using Math.random() for sensitive data (invite codes and shortlinks).\n**Learning:** Generating codes via Math.random() is predictable and easily exploitable. This poses a severe risk when it comes to access controls and invites.\n**Prevention:** Use a CSPRNG (Cryptographically Secure Pseudo-Random Number Generator) like node:crypto's randomInt or randomBytes for generating security-related tokens.\n
